Maîtriser la Radio et l'Accéléromètre en Python

Salut ! Ce guide va t'expliquer comment faire communiquer deux cartes micro:bit en utilisant les mouvements de ta main. C'est plus simple qu'il n'y paraît !

settings_input_antenna 1. Le concept : Émetteur vs Récepteur

L'Émetteur (La Manette)

Il lit l'inclinaison via l'accéléromètre et envoie un signal radio.

Le Récepteur (Le Robot)

Il écoute et déclenche une action (image, moteur) quand il reçoit le signal.

tune 2. Configuration de la Radio

Pour que tes cartes se parlent, elles doivent être sur la même fréquence.

  • Le Canal (channel) : Ta fréquence (0 à 100). Choisis un numéro unique pour ne pas capter tes voisins !
  • Le Groupe (group) : Une sous-adresse (0 à 255) pour plus de sécurité.
import radio
radio.on()
radio.config(channel=42, group=1)

3d_rotation 3. L'Accéléromètre

L'accéléromètre mesure les forces sur 3 axes. Les valeurs vont de -1024 à 1024.

X
Gauche / Droite
Y
Avant / Arrière
Z
Haut / Bas

code 4. Code de la Manette (Émetteur)

On programme la carte pour envoyer des mots-clés selon l'inclinaison.

from microbit import *
import radio

radio.on()
radio.config(channel=7, group=1)

while True:
    inclinaison = accelerometer.get_y()
    if inclinaison < -300:
        radio.send("AVANT")
        display.show(Image.ARROW_N)
    elif inclinaison > 300:
        radio.send("ARRIERE")
        display.show(Image.ARROW_S)
    else:
        radio.send("STOP")
        display.clear()
    sleep(100)

lightbulb 6. Astuces de Pro

🚀 Booster la portée : Utilise radio.config(power=7) pour envoyer le signal au maximum.
⚠️ Éviter les lags : Ne mets pas de sleep() trop long sur le récepteur, sinon il va rater des messages.
🔍 Debug : Utilise display.scroll() pour voir ce que le robot reçoit réellement.
