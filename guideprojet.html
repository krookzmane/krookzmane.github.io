🚀 Piloter par Radio avec l'Accéléromètre

Le guide interactif pour tes projets Micro:bit

1. C'est quoi le plan ?

Tu vas transformer ton micro:bit en manette de jeu. Quand tu penches la carte, elle envoie un signal invisible (ondes radio) à une autre carte qui va réagir.

  • Canal : Doit être le même sur les deux cartes (ex: 42).
  • Vitesse : On envoie les infos toutes les 100ms pour que ce soit fluide.

2. Teste l'inclinaison (Simulateur)

Bouge les curseurs pour voir comment l'accéléromètre réagit et quel message serait envoyé par radio.

3. Le Code de l'Émetteur (Ta Manette)

Copie ce code dans l'éditeur Python de ton micro:bit :

from microbit import *
import radio

radio.on()
radio.config(channel=42) # Choisis ton canal favori

while True:
    y = accelerometer.get_y()
    if y < -300:
        radio.send("AVANT")
        display.show(Image.ARROW_N)
    elif y > 300:
        radio.send("ARRIERE")
        display.show(Image.ARROW_S)
    else:
        radio.send("STOP")
        display.clear()
    sleep(100)

4. Le Code du Récepteur (Ton Robot)

from microbit import *
import radio

radio.on()
radio.config(channel=42)

while True:
    msg = radio.receive()
    if msg == "AVANT":
        display.show("A")
    elif msg == "ARRIERE":
        display.show("R")
    elif msg == "STOP":
        display.clear()
